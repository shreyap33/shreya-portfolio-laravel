# 🌸 Shreya Portfolio

A modern Pinterest-style aesthetic portfolio built using Laravel and JavaScript.

## ✨ Features
- Laravel backend
- JavaScript frontend interactions
- Photography & videography showcase
- Video editing portfolio
- Story writing section
- Cypress testing

## 🛠 Tech Stack
- Laravel
- PHP
- JavaScript
- Cypress

## 🚀 Setup

```bash
git clone https://github.com/your-username/shreya-portfolio-laravel.git
cd shreya-portfolio-laravel
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
